# 前言

sizeos:使用nwjs和vue搭建桌面系统

## 技术栈
vue2  + vue-router + webpack + ES6/7 + less + nwjs(webview)

## 项目运行

```

cd sizeos
npm install
npm install nwjs -g
npm run build && npm run native

```
